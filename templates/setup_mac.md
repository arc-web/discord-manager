# 💻 Mac Setup - Step by Step

**Step 1: Open your terminal**
- Click the magnifying glass icon in the top right corner of your screen (or press Command + Spacebar)
- Type **terminal**
- Click on **Terminal** when it appears

**Step 2: Install Claude Code**
Copy and paste this line, then hit Enter:
```
curl -fsSL https://claude.ai/install.sh | sh
```

**Step 3: Set your API key**
```
export ANTHROPIC_API_KEY={{api_key}}
```

**Step 4: Start Claude Code**
```
claude
```

If Claude responds, you're in! Raise your hand.
