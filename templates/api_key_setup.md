# 🔑 Pod {{pod_letter}} - API Key Setup

**Mac/Linux:**
```
export ANTHROPIC_API_KEY={{api_key}}
```

**Windows PowerShell:**
```
$env:ANTHROPIC_API_KEY="{{api_key}}"
```

Then type `claude` and hit Enter.

**How to tell which one you are:** If your terminal starts with `PS C:\` you're on Windows PowerShell. If it shows `%` or `$` you're on Mac/Linux.
